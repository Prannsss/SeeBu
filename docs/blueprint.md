# **App Name**: SeeBu Landing

## Core Features:

- Interactive Hero Section: Visually appealing hero section with clear calls to action and a Cebu City-themed graphic to capture user attention immediately.
- Step-by-Step Guide: Simple 3-step process (submit, LGU process, track status) explained with icons and concise descriptions for ease of understanding.
- Feature Highlights: Card-based feature listing showcasing real-time tracking, geo-tagged reporting, and service requests with mini UI previews.
- Benefits Overview: Highlight the key benefits for both citizens and the LGU using clear icons and direct statements emphasizing faster response and transparency.
- Call to Action: Enticing call to action to encourage users to join the platform, making them feel part of the solution for a better Cebu.
- Login Page: User authentication page allowing existing users to securely access their accounts.
- Sign-Up Page: New user registration page, collecting necessary information and initiating the email verification process.
- Email Verification: Process to verify the user's email address during registration, ensuring account security and communication.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke a sense of trust and civic duty.
- Background color: Very light blue (#E3F2FD) provides a clean and airy feel.
- Accent color: Soft green (#A5D6A7) for CTAs and highlights, suggesting growth and improvement.
- Font: 'Inter', a sans-serif font, should be used for both headlines and body text. Its neutral and modern design helps in creating a clean and accessible layout.
- Use a consistent 8px grid system for spacing and alignment across the landing page. Ensure mobile responsiveness.
- Employ clean, simple line icons to represent the various features and steps, enhancing visual communication.
- Subtle fade-in animations on scroll and hover effects on buttons to add a touch of interactivity without overwhelming the user.
